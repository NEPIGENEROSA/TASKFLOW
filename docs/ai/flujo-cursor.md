# Flujo de Trabajo con Cursor AI

Mi proceso de trabajo se dividió en tres etapas:
1. **Predicción y Autocompletado:** Uso de la IA para escribir funciones repetitivas de forma rápida.
2. **Edición en Bloque (Ctrl+K):** Seleccioné funciones enteras para pedirle a la IA que añadiera comentarios JSDoc y mejorara la legibilidad.
3. **Composer (Ctrl+I):** Utilicé el chat avanzado para coordinar cambios entre el HTML y el JS al añadir la función de búsqueda.
