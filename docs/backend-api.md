# Conceptos Clave de Backend y APIs

* **Axios**: Librería para realizar peticiones HTTP asíncronas. Es preferida sobre `fetch` porque maneja automáticamente la transformación de JSON y tiene mejor gestión de errores.
* **Postman**: Herramienta fundamental para testear endpoints sin necesidad de interfaz gráfica. Permite verificar códigos de estado (201, 404, 500).
* **Sentry**: Plataforma de monitoreo que captura errores en tiempo real en entornos de producción, enviando alertas detalladas a los desarrolladores.
* **Swagger**: Herramienta de documentación que genera una interfaz interactiva para que otros desarrolladores entiendan cómo consumir la API.
